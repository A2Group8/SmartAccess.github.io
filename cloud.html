Clouds, Services and Servers

What does it do?

As time passes, technology has become one of the main topics in the world. With technology many things can be achieve, for instance, storing data without using any USB flash drive or external hard disk. The data will now be stored in the cloud server instead. A cloud server is logical server that is hosted and delivered through a cloud computing platform such as Amazon Web Services (AWS) over the internet. Cloud servers has the similar functions and capabilities to a common server. However, cloud servers can be access casually from a cloud service provider. Cloud server is also known to be a virtual server or virtual private sever. Since cloud server is still a new way of storing data, many people are not comfortable with it yet. Hence, as time passes people will be more depended on it. For now, the cloud server is quite expensive compared to a normal storing device. Thus, making the price affordable will be a wise decision. Company can create a deal such a buying this much of gigabyte will give some free amount of gigabyte too. By doing this, people will be more attracted to the cloud company. Besides, the company can also advertise their purpose of storing the data, photos and etc. in the cloud server. For instance, people store their data in a physical storing device and they happen to loss it. Thus, they can retrieve back the data they lost. On the other hand, if they store the data in the cloud, the data will not be going anyway unless it is deleted in the cloud itself. In few years’ time, the cloud server will be much cheaper compared to current cloud server. Thus, the market of purchasing cloud server will increase and this will lead the people to store their data in the cloud instead of a physical storing device. A cloud computing technology is the computing components such as hardware, software and infrastructure that enable the delivery of cloud computing services such as software as a service (SaaS), platform as a service (PaaS) and Infrastructure as service (IaaS) via network which is internet. Additionally, cloud computing relies on the resources that had been shared to achieve logical and economies of scale.

What is the likely impact?

Cloud computing can potentially affect the business as moving the business to the cloud is always thought to be the ending of the business, yet rather the beginning of it. During this period of migration, the cloud computing impact has a wave effects on the internal business operations and processes. Hence, it is crucial not to overlook these ongoing changes as with any changes to the organisation, the business flow and processes will be altered in numerous ways that are unexpected. For instance, the efficiency of the business flow will be slow down as it is attempting to a new setup. Not only that, there are likely to be unpredictable consequences that will affect the internal effort of an organisation. For instance, the team might not have the skills necessary to run the new environment which might cause inefficient work. The people that will be affected the most will be the employers. These changes will force them to learn about the cloud systems which might be easier for newer generation as they have been exposed to technologies at a very young age. On the other hand, older generation will be suffering as they had to remember the steps and sequent to manage the cloud. Finally, this will create more job opportunity in the IT department as they will be hiring more IT students to handle the cloud network. As for normal jobs it would stay the same.

How will this affect you?

Cloud computing became an essential as I can store my important documents or photos without being afraid of losing it. Not only that, it can be easily accessed by any devices if I have the account and password right. With cloud server I don’t have to spend time on transferring data from computer to a physical device. As I have been through the era with a physical storing device and the digital storing device, I would prefer digital storing device. This is because if I lose my physical storing device I lose everything in it and I can’t retrieve the documents or photos inside. As for my family, I am sure the siblings will be easily be attempted to it since they are familiar with basic IT skills. For parents, it is difficult as they are not that keen on learning IT skills as they don’t find it a need for them to learn cloud computing and other IT skills.
