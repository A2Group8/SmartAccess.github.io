What does CyberSecurity do?

Technology has advanced rapidly over the last few decades to a point where almost every business depends on a computer system of that stores and track valuable data. The world wide web has made running business much more efficient. However, it also made companies more valuable to cyber theft. Which is why cybersecurity has become necessity and has high demand in the modern era. Cybersecurity is the protection of internet-connected systems that includes hardware, software and data from any cyberattacks. Cybersecurity are used by enterprises to protect against any unauthorized access to data centres and other computerized systems. Information security which is designed to maintain the confidentiality, integrity and availability of data, is a subset of cybersecurity. To ensure cybersecurity, which is required the coordination of efforts throughout an information system that includes the application security, information security, network security, the recovery system, operation security, and end-user education. Keeping the system safe can be a challenging task especially when cybersecurity is constantly evolving and must keep up with new technologies, security trends and threat intelligence. Cybersecurity threats can in different type of forms such as Ransomware, social engineering, phishing, and one of the famous most recognizable is Malware that is a file/program used to harm a computer user such as computer viruses, spyware and worms. Cybersecurity can help by preventing cyberattacks, data breaches and identity theft. A company/business must have a strong sense of security network and an effective incident plan because it is better to prevent any attack which could be a huge lost. Education is the key for prevention, ensuring that every user is aware of the potential treats they could face, whether it’s an email or insecure network, could be the difference between getting hacked and avoiding the risk altogether. For security purposes, users should update their password regularly. Strong password are one of the first line defences against breaches and changing them occasionally may help to prevent hackers. External application like Virtual private network could a useful tool as it creates a safe and encrypted connection over less secure network which means its hard for hackers to locate the user location. Purchasing anti-virus such as Norton security, total av or scanguard can is mandatory to keep devices safe from any leaks and attacks

What is the likely impact?

The worst way to find out is not to be secure. There are multiple factors that are after the user personal information and data which all of them are planned to use the information to strengthen their existing hold over the user life or illegally benefit without the user consent. Cybercriminals will use all kinds of methods to endanger the user such as names, savings, property, location, reputation which can cause both hackers and fake companies to target the user with cheap sale which was not advertised by the origin company or use the user identify by force to trick other people. One of the common attacks on the world wide web is after the user personal information. Cybercriminals starts and ends with governments, corporations and hacker trying to obtain the user name, social security number, credit card information and others that can be used for identity theft or other benefits. The data that store inside the device not necessarily safe as it can be hacker as well. It should be viewed as a separated target than the user personal information, mainly as hackers do not need to who the user is to misuse the data. There are two things hackers are looking for when have the chance to gain the user data. One of them are ransomware which is a type of malicious software designed to block access to the computer system until a sum of money is paid. The second is called catfish food where it uses fake profile picture to scam other users to send money and personal information, this is highly common in the dating websites where cybercriminals used young women picture to lure men into the trap. How will cybersecurity affect you?

Technologies devices are such a big part in our daily life and it is hard to imagine without them. The use of technology can be used to keep in touch, pay bills, stay updated on news, shopping, research for information. However, our data can be exposed to criminals or cybercriminals. As mentioned before, the main purpose of cybersecurity is to protect from cyberattacks. Without cybersecurity, the risk of losing all valuable data is extremely high. Malware includes nuisances like spyware which allows digital hackers to track every move and to view the passwords. Typically, more than thousands of people are tricked into downloading malware by accident and when its downloaded, a rogue website or a free software advertisement will appear like a screensaver. This is where cybercriminals will remotely take over control of the user data. This can be spread by friends on social networking like “Facebook”. Another common way is sending a fake advertisement to the user email which is called “SPAM” that will tricked the user by sending an email containing links or attachment promoting “free programs” to download a computer virus. This can be used to defraud to the user social connection. For instance, the criminal hacked into the user account and send a message to every user contact available to wire money or send threats to the user family and friends. Knowing all possible treats is crucial as to protect user data and keeping family and friends from harm.
